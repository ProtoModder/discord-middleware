⚙️ Discord Middleware Admin

Configure server filters, security settings, and middleware behavior

📖 Connection Guide

1️⃣ How to Connect Middleware to Discord

Step 1: Create Discord Bot
  1. Go to Discord Developer Portal
  2. Click "New Application" and give it a name
  3. Go to "Bot" section in the sidebar
  4. Click "Reset Token" to get your bot token
Step 2: Get Your Token
  • Copy the bot token (keep it secret!)
  • You'll enter this in the middleware config
Step 3: Add Bot to Server
  1. In Developer Portal, go to "OAuth2" → "URL Generator"
  2. Select scopes: bot
  3. Select permissions: Read Messages/View Channels, Send Messages
  4. Copy the generated URL and open it in your browser
  5. Select your server and authorize the bot
Step 4: Configure in UI
  1. Add your Server ID in the Server Management section
  2. Add the channel IDs you want to monitor
  3. Set your bot token in config.yaml
  4. Save configuration

2️⃣ How to Connect OpenClaw to Middleware

OpenClaw sends messages to the middleware endpoint, which processes them and returns responses back to OpenClaw.

Configuration

In your OpenClaw config, set the middleware URL:

middleware_url: http://localhost:3000/webhook
How It Works
  1. OpenClaw sends a message to the middleware endpoint
  2. Middleware processes the message (applies filters, security checks)
  3. Middleware returns the processed response to OpenClaw
  4. OpenClaw continues with the response
Environment Variables 
DISORD_BOT_TOKEN=your_bot_token_here
MIDDLEWARE_PORT=3000

🚀 Quick Start Guide

1. Copy config file 
cp config.yaml.example config.yaml
2. Add Server ID
  • Enable Developer Mode in Discord (User Settings → Advanced)
  • Right-click your server name → "Copy ID"
  • Add it in Server Management above
3. Add Channel IDs
  • Right-click any channel → "Copy ID"
  • Add channel IDs to filter specific channels
4. Start Middleware 
# Activate virtual environment
source venv/bin/activate

# Run the middleware
python middleware.py

🤖 How to Connect Your AI

Option A: Ollama (Local AI)
  1. Install Ollama from ollama.com
  2. Run a model: ollama run llama2
  3. Set in config.yaml:
llm_endpoint: "http://localhost:11434/api/generate"
llm_api_key: ""
Option B: OpenAI
  1. Get API key from OpenAI Platform
  2. Set in config.yaml:
llm_endpoint: "https://api.openai.com/v1/chat/completions"
llm_api_key: "sk-your-key-here"
Option C: OpenClaw (Existing AI)
  1. Set in config.yaml:
openclaw_url: "http://localhost:8080"

Messages will forward to your existing OpenClaw setup.

🔍 How to Find Server & Channel IDs

Enable Developer Mode
  1. Open Discord
  2. User Settings → Advanced → Developer Mode: ON
Copy Server ID
  1. Right-click your server name (top of channel list)
  2. Click "Copy Server ID"
  3. Paste in Server Management section above
Copy Channel ID
  1. Right-click any channel in your server
  2. Click "Copy Channel ID"
  3. Paste in config or add via UI
